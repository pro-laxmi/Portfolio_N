### Project Overview

Developed an offline multimodal Retrieval-Augmented Generation (RAG) system supporting semantic search over PDFs, audio transcripts, and images using ChromaDB and FAISS HNSW indexing. Implemented natural-language image retrieval using CLIP embeddings and cosine similarity search, enabling top-k image retrieval from local datasets. Integrated Whisper and locally hosted Phi-3 models via Ollama to perform retrieval-augmented question answering with fully offline inference.

This project demonstrates my ability to tackle complex engineering problems, apply rigorous mathematical and scientific principles, and build highly performant systems.
