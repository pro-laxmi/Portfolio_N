### Project Overview

Developed GPU-accelerated 0D and 1D plasma simulation modules using PyTorch tensors and Numba, enabling efficient large-scale numerical computations. Implemented transport solvers using finite-difference methods and Scharfetter-Gummel discretisation to model particle transport and plasma dynamics. Designed a modular object-oriented simulation framework supporting configurable physics models, parallel computations, and extensible scientific workflows.

This project demonstrates my ability to tackle complex engineering problems, apply rigorous mathematical and scientific principles, and build highly performant systems.
