### Project Overview

Built a high-throughput semantic search pipeline using Qdrant and Apache Solr, enabling sub-second retrieval across 47M+ articles from library article corpus. Implemented Int8 vector quantisation to solve OOM constraints, reducing the memory footprint by 75% (250GB to 63GB) while optimising the HNSW graph index.

This project demonstrates my ability to tackle complex engineering problems, apply rigorous mathematical and scientific principles, and build highly performant systems.
