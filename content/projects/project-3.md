### Project Overview

Architected a hybrid database engine utilizing Shannon Entropy to autonomously infer schemas from raw JSON, dynamically routing flat data to SQL columns and nested structures to JSONB. Engineered a logical query translator supporting dynamic LEFT JOINs and ACID transactions, achieving 5.9 ms average query latency and reducing multi-entity write latency by 38.8% vs. direct sequential SQL.

This project demonstrates my ability to tackle complex engineering problems, apply rigorous mathematical and scientific principles, and build highly performant systems.
